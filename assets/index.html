What is it?

Everyone loves the Google CDN right? Even Microsoft runs their own CDN.

The problem is, they only host the most popular libraries.

We host the other stuff.


Change it

Our goal is to operate this CDN in a peer reviewed fashion.

This means only the highest quality libraries vetted by the community get added to cdnjs.com

Use the feedback button to have your say.

Libraries

  • Backbone.js (0.3.3)

    Backbone supplies structure to JavaScript-heavy applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing application over a RESTful JSON interface.

  • CamanJS (2.0)

    Pure Javascript (Ca)nvas (Man)ipulation.

  • HeadJs (0.9)

    Load scripts like images. Use HTML5 and CSS3 safely. Target CSS for different screens, paths, states and browsers. Make it the only script in your HEAD

  • jQuery (1.4.4) - Hosted by Google

    A fast, concise, library that simplifies how to traverse HTML documents, handle events, perform animations, and add AJAX.

  • LABjs (1.2.0)

    The core purpose of LABjs is to be an all-purpose, on-demand JavaScript loader, capable of loading any JavaScript resource, from any location, into any page, at any time.

  • less.js (1.0.41)

    LESS extends CSS with dynamic behavior such as variables, mixins, operations and functions. LESS runs on both the client-side (IE 6+, Webkit, Firefox) and server-side, with Node.js.

  • json2.js

    This file creates a JSON property in the global object, if there isn't already one, setting its value to an object containing a stringify method and a parse method.

  • Modernizr (1.6)

    Modernizr adds classes to the element which allow you to target specific browser functionality in your stylesheet. You don't actually need to write any Javascript to use it.

  • Raphaël (1.5.2)

    Raphaël is a small JavaScript library that should simplify your work with vector graphics on the web. If you want to create your own specific chart or image crop and rotate widget, for example, you can achieve it simply and easily with this library.

  • Require.js (0.23.0)

    RequireJS is a JavaScript file and module loader. It is optimized for in-browser use, but it can be used in other JavaScript environments, like Rhino and Node.

  • sammy.js (0.6.8)

    A small web framework with class.

  • $script.js (1.1)

    $script.js is an asynchronous JavaScript loader and dependency manager with an astonishingly impressive lightweight footprint by Justin Diaz

  • SizzleJs (1.4.4)

    A pure-JavaScript CSS selector engine designed to be easily dropped in to a host library.

  • Underscore.js (1.1.4)

    Underscore is a utility-belt library for JavaScript that provides a lot of the functional programming support that you would expect in Prototype.js (or Ruby), but without extending any of the built-in JavaScript objects. It's the tie to go along with jQuery's tux.

  • yepnope.js (0.2.3)

    yepnope is a small wrapper around the super-fast LABjs script loader, and it allows you to load only the scripts that your users need.

  • xui.js (2.0.0)

    a super micro tiny dom library for authoring html5 mobile web applications.

Our goals


  • Make the web faster
    • Provide CDN hosting (Amazon Cloudfront) for popular javascripts not provided by Google APIs
    • Serve minified, gzipped copies of scripts
    • Use a fast, reliable DNS service (Amazon Route 53)

  • Make web development easy
    • Want to use your favourite script? Easy. Just use the pre-generated script tags provided by cdnjs. No downloading, no ftp, no worries!

  • Get your scripts noticed
    • Just wrote a cool new library? Easy. Just push it to cdnjs. Now everyone has fast, easy access to it. (TODO)

  • Promote the use of Javascript
    • It’s a really cool language!
